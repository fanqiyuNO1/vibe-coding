# 猎巫镇 DM 语音播报器 — 开发功能

## Context
替代《猎巫镇》桌游中的人工 DM，语音播报黎明和夜晚阶段的流程指令。**核心是语音播报，不是提词器**。播报前可编辑台词和等待时间，确认无误后开始播报，播报中纯语音，不显示文字。

## 二、播报流程

### 黎明阶段（5 句）
| # | 台词 | 等待 |
|---|------|------|
| 1 | 所有玩家请闭眼 | 3 秒 |
| 2 | 女巫请睁眼，互相确认身份 | 3 秒 |
| 3 | 女巫请将黑猫放在任意一名玩家面前 | **5 秒** |
| 4 | 女巫请闭眼 | 3 秒 |
| 5 | 所有玩家请睁眼 | 3 秒 |

### 夜晚阶段（7 句）
| # | 台词 | 等待 |
|---|------|------|
| 1 | 所有玩家请闭眼 | 3 秒 |
| 2 | 女巫请睁眼，选择今晚要杀害的角色 | **10 秒** |
| 3 | 女巫请闭眼 | 3 秒 |
| 4 | 警长请睁眼 | 3 秒 |
| 5 | 警长请选择要保护的人 | **5 秒** |
| 6 | 警长请闭眼 | 3 秒 |
| 7 | 所有玩家请睁眼 | 3 秒 |

---

## 三、交互逻辑

- 首页 → 阶段选择 → 编辑页（可编辑台词和等待秒数） → 开始播报 → 纯语音
- 编辑页：每句台词可自由编辑，每句等待秒数独立设置
- 播报中显示「播报中...」，不可中断
- 播报结束后自动返回阶段选择页，可再次选择阶段播报
- 黎明、夜晚均可重复触发
- **播报中不显示任何台词，纯语音**

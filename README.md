# 🧙 猎巫镇 DM 语音播报器

> Salem 1692 · 桌游主持人语音助手 · 浏览器原生 TTS

一款专为**猎巫镇 / 狼人杀类桌游**设计的主持人语音播报工具，基于浏览器内置语音引擎，零依赖、开箱即用。

🔗 **在线使用** 👉 [fanqiyuno1.github.io/vibe-coding/salem-dm](https://fanqiyuno1.github.io/vibe-coding/salem-dm/)

---

## ✨ 特性

- 🎙️ **语音播报** — 调用浏览器原生 TTS 引擎，中文女声朗读，语速适中
- ✏️ **台词可编辑** — 黎明 5 句、夜晚 7 句台词均可自由修改，等待秒数可调
- 👆 **交互选人** — 夜晚阶段弹出编号面板，女巫点选杀害目标，脉冲动画反馈
- 🃏 **翻牌揭示** — 播报结束后 3D 翻转卡片揭示结果，仪式感拉满
- 📱 **移动端适配** — 模拟手机外观，刘海屏、安全区、小屏断点全覆盖
- ⚡ **零依赖** — 纯 HTML/CSS/JS 单文件，无需安装任何东西

---

## 🚀 快速开始

```bash
# 方式一：直接用浏览器打开
open index.html

# 方式二：本地 HTTP 服务器（推荐，确保语音引擎正常）
npx serve .
```

> ⚠️ Chrome 等浏览器要求 HTTPS 才能使用 SpeechSynthesis API，本地开发可在 `localhost` 下正常运行。部署到线上请使用 GitHub Pages / Vercel 等自动提供 HTTPS 的平台。

| 技术 | 用途 |
